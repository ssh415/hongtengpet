**宏腾宠物之家**--后台管理平台

一、功能模块：

1、账号管理

`（1）账号管理`

`（2）账号登录、退出`

`（3）个人信息编辑`

`（4）角色管理`

2、宠物管理

`（1）宠物新增：品种，进货地，性别，年龄，毛色，简介，图片……`

`（2）信息编辑`

`（3）货存管理`

`（4）品种管理：大类、小类`

3、订单管理

`（1）创建订单：包括买家下单和自己采购下单`

`（2）订单列表：买家订单信息`

`（3）物流信息：买家订单物流`

`（4）采购管理：采购历史订单`

二、开发技术

1、后台：

`（1）web框架：express`

`（2）ORM框架：sequelize`

`（3）日志：morgan，log4js`

2、DB：

`（1）MySQL`

3、前端：

`（1）UI框架：layui`

`（2）ajax：axios`
